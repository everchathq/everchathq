## Roadmap

### Stage 1 - MVP
Users are able to do the basics.  
**Release:** May 2021

- Authentication
- CRUD channels
- CRUD posts
- Image
- GIF

<br/>

### Stage 2 - Discord Killer
Users are able to interact with each other in better ways.  
**Release:** November 2021

- Live stream
- Conference call

<br/>

### Stage 3 - Media Upgrade
Users are able to upload any type of media.  
**Release:** Unspecified

- Markdown
- Voice recording
- Code snippet
- QR code
- Sticker API
- GIF API

<br/>

### Stage 4 - Politics
Users are able to govern communities.  
**Release:** Unspecified

- Stand for election
- Propose community laws
- Vote
